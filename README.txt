The Robot Framework Oracle Library is a library which provides common functionality for testing Oracle database contents. This library like Robot Framework DatabaseLibrary, but this library provide set connection encoding (NLS_LANG) and execute PL/SQL scripts. See OracleLibrary.html for more information about provided keywords.
